## ðŸ†˜ Troubleshooting

### "Invalid API Key" Error
- Double-check you copied the full key
- Make sure there are no extra spaces
- Verify the key is for the correct provider
- Check if the key is still active

### "Rate Limit Exceeded"
- You've hit the daily/minute limit
- Wait a few minutes or try the next day
- Consider switching to another provider
- Or create a second free account

### AI Not Working
- Check your internet connection
- Verify AI is enabled in Settings
- Make sure you saved your settings
- Try a different provider

### Still Need Help?
- Check provider's documentation
- Visit provider's support page
- Create an issue on GitHub
